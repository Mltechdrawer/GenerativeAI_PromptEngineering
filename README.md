# Generative AI + Prompt Engineering (MkDocs)

## Requisitos
- Python 3.x
- mkdocs + mkdocs-material

## Instalación
```bash
pip install mkdocs mkdocs-material pymdown-extensions
```

## Servir en local
```bash
mkdocs serve
```

## Construir
```bash
mkdocs build
```

## Publicación en GitHub Pages
Puedes usar GitHub Actions o `mkdocs gh-deploy`.
