# Block IV — Practical applications and personalization (45 min)

## Objective
Apply what has been learned to a real workflow and create a **personalized assistant** (Gems / GPTs / similar tools) for recurring tasks.

## Typical use cases

### Teaching
- Generation of rubrics, assignments, and model solutions.
- Feedback on submissions using fixed criteria.
- Materials: summaries, examples, questionnaires.

### Business
- Reports, meeting notes, emails, proposals.
- Document analysis and executive summaries.
- Process support (checklists, SOPs, templates).

### Software development
- Refactoring, test generation, documentation.
- Pull request reviews with quality criteria.
- Migrations and explanation of legacy code.

## Personalization (demo + practice)
- Define the objective of the assistant.
- Persistent instructions: tone, limits, format.
- Prompt library: “shortcuts” for frequent tasks.
- Evaluation: how to detect failures and improve.

## Final exercise
Create a “mini-assistant” for a real use case and prepare:

- Master prompt (template).
- 2 secondary prompts (variants).
- Output verification checklist.

> Expected outcome: something you can reuse tomorrow in your work.
