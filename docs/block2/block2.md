# Block II — Tools and current landscape (45 min)

## Objective
Understand the ecosystem of tools (ChatGPT, DALL·E, Sora, Gemini, Perplexity, Meta AI, Copilot…) and when to use each of them.

## Quick tools overview
- **Chatbots/Assistants**: ideation, writing, analysis, and general support.
- **Image generation**: sketches, illustrations, teaching materials, assets.
- **Video generation**: prototypes, visual concepts, storytelling.
- **Search with sources**: questions that require traceability/citations.
- **Code assistants**: generation, refactoring, tests, explanation, and documentation.

## Practical comparison (guided exercise)
The same objective, three approaches:

1. **“Explain it to me”** (conversational model)
2. **“Search with sources”** (citation-based engine)
3. **“Do it in code”** (programming assistant)

> Tip: it’s not about “a better tool”, but about “a more appropriate tool” depending on the context.

## Activity
Choose a real case (teaching, business, development) and answer:

- Which tool would you use?
- What output do you expect?
- What risks are involved (bias, confidentiality, accuracy)?
